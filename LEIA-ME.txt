PREFIX v2.1.2 - Renomeador de Arquivos
======================================

USO NORMAL
1. Abra dist\PREFIX.exe, quando o executavel ja estiver compilado.
2. Selecione uma pasta.
3. Digite o prefixo.
4. Revise a pre-visualizacao e confirme.
5. Use "Desfazer ultima acao" quando necessario.

COMPILAR E VALIDAR O .EXE
1. Instale qualquer versão do Python 3 no Windows. O script prioriza o Python 3.14 quando ele estiver disponível.
2. Execute gerar_executavel.bat.
3. O script cria um ambiente isolado, executa auditoria estatica,
   roda os testes, compila e executa o autoteste dentro do novo .exe.
4. O autoteste precisa gerar um marcador de sucesso; apenas o codigo de
   saida do processo nao e considerado suficiente.
5. Somente quando aparecer "APROVADO" use dist\PREFIX.exe.

SEGURANCA
- O aplicativo nao processa subpastas.
- Conflitos e nomes maiores que o limite do Windows nao sao alterados.
- Antes de renomear ou desfazer, e criado um diario oculto de recuperacao.
- Se o aplicativo ou computador for interrompido, selecione novamente a
  mesma pasta para recuperar o historico automaticamente.
- Em um desfazer parcial, os itens pendentes permanecem no historico.

ARQUIVOS PRINCIPAIS
- renomeador_prefixo.py: interface grafica.
- renomeador_core.py: regras, validacoes, diario e desfazer.
- tests\: testes automatizados.
- PREFIX.spec: configuracao do PyInstaller.
- gerar_executavel.bat: compilacao e validacao completa.
- AUDITORIA.md: relatorio tecnico da auditoria.
